Tarkov Price Overlay (Beta)
============================

설치
----
1. 받은 .exe 인스톨러를 더블클릭합니다.
2. Windows Defender SmartScreen이 "Windows에서 PC를 보호했습니다" 경고를
   표시하면:
     "추가 정보" → "실행" 을 누릅니다.
   (이는 코드 사이닝 인증서가 없는 베타 빌드라 정상입니다.)
3. 설치가 끝나면 시작메뉴에서 "Tarkov Price Overlay"를 실행합니다.

사용
----
- 게임 실행 후 아이템 위에 마우스를 올리고 F2 키를 눌러보세요.
- 가격 카드가 떠있는 동안 마우스를 카드에 올리면 사라지지 않습니다.
- 헤더(상단 제목 부분)를 잡고 드래그하면 위치를 바꿀 수 있습니다.
- 우측 상단 ⚙ 아이콘에서:
    * 언어 (한국어/English)
    * 게임 모드 (PVP/PVE)  ← 본인 게임 모드와 일치시키세요!
    * 단축키 변경 (F2 외 다른 키 가능)
    * 표시 시간 (몇 초 후 자동 사라짐)
    * 캡처 영역 미세 조정 (모달에서 슬라이더로 조정, 청록 십자선이 마우스 기준점)

주의 사항
----------
- 첫 실행 시 EasyOCR 모델 로딩으로 5~15초 정도 시작이 길 수 있습니다.
- 게임 해상도/배율에 따라 캡처 영역이 안 맞을 수 있습니다 — 그때는
  설정 → 캡처 영역 → 설정하기로 조절하세요.
- 가격 데이터는 https://tarkov.dev 의 공개 API를 사용합니다 (인터넷 필요).

문제 발생 시
-------------
- 콘솔 창(검은 창)에 찍히는 로그를 캡처해서 보내주세요.
- 작업 관리자에서 "tarkov-server.exe"가 떠있는지 확인해주세요.

베타 테스트에 참여해주셔서 감사합니다!
