Getting started

Three steps

Download & install

Installer or portable ZIP from Releases. First launch downloads the OCR model once (1–5 min).

Run as administrator

Tarkov runs elevated, so run the overlay as admin too — otherwise the hotkey won't reach the game.

Press F2 in-game

Make the item name visible, press F2. Drag any blank area of the card to move it (auto-saved).

Safety

Ban risk, honestly

Screen capture + OCR only — the same category as YouTube recorders or Discord screen share

❌ What it does NOT do

  • Read/write game process memory
  • Inject DLLs
  • Hook the game window / low-level keyboard
  • Modify any game files
  • Send your searches, screen, personal data, or IP

✅ What it does

  • Standard Win32 screen capture (BitBlt)
  • A transparent topmost overlay window
  • OS-level hotkey (RegisterHotKey)
  • Fetch prices from the tarkov.dev public API
  • Anonymous usage stats (optional, can be disabled)
FAQ

Frequently asked

Is it free?

Yes, completely free and MIT open-source. You can review the full code on GitHub and build it yourself.

Will I get banned?

It uses screen capture and OCR only; it doesn't read game memory or inject. It's a fundamentally different category from memory hacks and ESP. BSG hasn't officially sanctioned it, so use is at your own risk — and no bans have been reported for similar OCR-based overlays to date.

F2 does nothing inside the game.

Tarkov runs as administrator, so right-click the app → "Run as administrator" for the hotkey to reach the game. Also switch the game's graphics to "Borderless Fullscreen" (exclusive fullscreen blocks every overlay at the OS level). If it still fails, rebind the hotkey in Settings.

SmartScreen / antivirus warning on install.

The executable is unsigned for cost reasons, not malware. Click "More info → Run anyway". The behavior and full source are open on GitHub.

Dual monitor / 4K / 125% scaling — coordinates are off.

Per-monitor DPI is handled. If it's still off, fine-tune the capture X/Y offset in Settings, or hit "Auto-fit resolution".

PVE prices look wrong.

Switch Game mode to PVE in Settings. PVP and PVE price data are stored separately.
