안전성

밴 위험, 정직하게

화면 캡처 + 글자 인식(OCR)만 사용 — 유튜브 녹화·디스코드 화면공유와 같은 카테고리

🛡️ 밴에 걸리는 동작은 하나도 쓰지 않습니다 — 밴 위험 없이 쓰도록 만들었어요

메모리 핵·ESP·에임봇과는 완전히 다른 카테고리이고, 동일한 방식의 오버레이로 밴된 사례는 현재까지 보고된 바 없습니다. 다만 BSG가 공식적으로 허용한 도구는 아니라 100%까지 장담할 수는 없으며, 사용 책임은 본인에게 있습니다. 코드는 GitHub에 100% 공개되어 누구나 직접 검토할 수 있어요.

❌ 하지 않는 것

  • 게임 프로세스 메모리 읽기/쓰기
  • DLL 인젝션
  • 게임 윈도우 / 저수준 키보드 후킹
  • 게임 파일 수정
  • 검색 내용·화면·개인정보·IP 전송

✅ 하는 것

  • 일반 Win32 화면 캡처(BitBlt)
  • 화면 위 투명 오버레이 윈도우
  • OS 표준 단축키(RegisterHotKey)
  • tarkov.dev 공개 API로 시세 조회
  • 익명 사용 통계(선택, 끄기 가능)
FAQ

자주 묻는 질문

무료인가요?

네, 완전 무료이며 MIT 라이선스 오픈소스입니다. GitHub에서 전체 코드를 확인하고 직접 빌드할 수도 있습니다.

이거 쓰면 밴당하나요?

화면 캡처와 OCR만 사용하고 게임 메모리를 읽거나 인젝션하지 않습니다. 메모리 핵·ESP와는 완전히 다른 카테고리입니다. 다만 BSG가 공식 허용한 도구는 아니므로 사용 책임은 본인에게 있고, 동일 방식 오버레이로 밴된 사례는 현재까지 보고된 바 없습니다.

F2를 눌러도 게임 안에서 반응이 없어요.

타르코프는 관리자 권한으로 실행되므로, 본 앱도 우클릭 → "관리자 권한으로 실행"해야 단축키가 전달됩니다. 또한 그래픽 설정을 "테두리 없는 전체화면"으로 바꿔주세요(독점 전체화면은 모든 오버레이를 OS가 차단). 그래도 안 되면 설정에서 단축키를 다른 키로 변경.

설치할 때 SmartScreen / 백신 경고가 떠요.

코드 서명 인증서 비용 문제로 서명을 안 했을 뿐 악성코드가 아닙니다. "추가 정보" → "실행"으로 진행하세요. 동작 원리와 전체 코드는 공개되어 있습니다.

듀얼 모니터 / 4K / 125% 배율인데 좌표가 어긋나요.

Per-monitor DPI가 처리되어 동작합니다. 그래도 어긋나면 설정 → 캡처 영역에서 X/Y 오프셋을 미세 조정하거나 "해상도 자동 조정"을 누르세요.

PVE인데 가격이 이상해요.

설정 → 게임 모드를 PVE로 변경하세요. PVP와 PVE 시세 데이터가 분리되어 있습니다.
