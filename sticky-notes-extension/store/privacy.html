스티키 메모 개인정보처리방침

최종 수정일: 2026-07-08

"스티키 메모"(이하 "본 확장 프로그램")는 사용자의 개인정보를 수집하지 않습니다.

1. 데이터 저장

사용자가 작성한 메모(텍스트, 위치, 크기, 색상, 표시 범위)와 확장 설정은 Chrome이 제공하는 로컬 저장소(chrome.storage.local)에만 저장됩니다. 모든 데이터는 사용자의 기기(브라우저 프로필) 안에만 존재합니다.

2. 데이터 전송

본 확장 프로그램은 어떠한 데이터도 외부 서버로 전송하지 않습니다. 네트워크 요청을 일절 수행하지 않으며, 분석(analytics)·광고·추적 도구를 포함하지 않습니다.

3. 페이지 콘텐츠 접근

본 확장 프로그램의 콘텐츠 스크립트는 메모를 화면에 표시하기 위해서만 각 페이지에서 실행되며, 페이지의 내용을 읽거나 수집하거나 수정하지 않습니다.

4. 데이터 삭제

각 메모는 메모의 삭제 버튼으로 언제든 삭제할 수 있습니다. 확장 프로그램을 제거하면 저장된 모든 데이터가 함께 삭제됩니다.

Sticky Notes Privacy Policy

Last updated: 2026-07-08

Sticky Notes ("the extension") does not collect any personal information.

1. Data storage

Notes you write (text, position, size, color, visibility scope) and extension settings are stored only in Chrome's local storage (chrome.storage.local). All data stays on your device, inside your browser profile.

2. Data transmission

The extension never transmits any data to external servers. It makes no network requests and contains no analytics, advertising, or tracking components.

3. Access to page content

The extension's content script runs on pages only to display your notes. It does not read, collect, or modify the content of the pages you visit.

4. Data deletion

Each note can be deleted at any time with its delete button. Uninstalling the extension removes all stored data.
